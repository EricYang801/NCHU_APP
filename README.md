# NCHU APP

### 部分為AI生成

NCHU APP 是一個專為中興大學學生開發的 iOS 應用程式，提供便捷的校園服務整合平台。

## 功能特點

- 🔐 整合中興大學 LMS 系統登入
- 📝 作業查詢與繳交
- 🔍 驗證碼自動識別
- 🎨 現代化使用者介面
- 🌙 深色模式支援

## 專案結構

```
NCHU_APP/
├── Assets.xcassets/  # 資源檔案
├── Components/       # 可重用元件
├── Models/          # 資料模型
├── Views/           # 視圖元件
├── Managers/        # 管理類別
├── Theme/           # 主題設定
├── Errors/          # 錯誤處理
├── NCHULMSLogin.swift    # LMS登入處理
├── OCRCaptcha.swift      # 驗證碼識別
├── ContentView.swift     # 主要視圖
└── NCHU_APPApp.swift     # 應用程式入口
```
